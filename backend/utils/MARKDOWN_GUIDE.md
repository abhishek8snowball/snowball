### **1. Headings (Use # symbols)**
```markdown
# Main Title (H1)

```

### **2. Text Formatting**
```markdown
**Bold text** or __bold text__
*Italic text* or _italic text_
***Bold and italic***
~~Strikethrough text~~
```

### **3. Lists**
```markdown
# Bullet Points
- First item
- Second item
- Third item

# Numbered Lists
1. First step
2. Second step
3. Third step

# Nested Lists
- Main item
  - Sub item
  - Another sub item
- Another main item
```

### **4. Links and Images**
```markdown
[Link text](https://example.com)
![Image alt text](image-url.jpg)
```

### **5. Code and Technical Content**
```markdown
`Inline code` for small snippets

```javascript

  console.log("Hello World!");
}
```
```

### **6. Blockquotes**
```markdown
> This is a blockquote
> It can span multiple lines
> Perfect for highlighting important information
```
